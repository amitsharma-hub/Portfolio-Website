## 📬 Contact Form

The contact form is integrated using **EmailJS**.

To make it work with your own account:

1. Create an account on EmailJS.
2. Create a Service.
3. Create an Email Template.
4. Replace the following values in `script.js`:

* Public Key
* Service ID
* Template ID
